What you can / can't do

Not sure when you'd need to use my logo's but in a case you do, make sure you follow the guidelines.

check_box

Use the brand when following guidelines

As long as you're following the guidelines, you can use my brand if required for something. If you do use it, drop me an email - i'm interested.

 check_box

Swap out the main wordmark with supplied colours

...in the appropriate conditions, in a mainly blue background, for example, swap out the main wordmark with the supplied grey variation.

 check_box

Give the brand some breathing room

Appropriate space should be placed around the logo or icon to ensure it properly stands out.

 highlight_off

Do not alter the logo in any way

Including (but not limited to) adding drop shadows, gradients, changing colours, stretching etc.

 highlight_off

Do not show a sign of promotion

Unless sponsored, don't lock up my wordmark or icon in a way that may indicate a promotion going on.

 highlight_off

Do not modify or remove any text

Text shown in the logo should be left as is without any modification.

 highlight_off

Do not cover or obscure the logo

The plexion wordmark or icon should be clearly visible to ensure it serves it's purpose.
