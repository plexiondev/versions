Grand Scheme of Things

The Grand Scheme of Things, "Haunting Hunters" was a group project to create a fun and unique take on the classic Hide & Seek formula. Hide inside or outside the great mansion or seek to find and kill the hiders. Use a range of powerups to your advantage, bought before the game, to help you score the victory for your team.

Installation & Setup

As with any other map, there is an incredibly simple installation process:

  1. Download the map .zip file and save it somewhere
  2. On Windows, press WIN + R and enter %appdata%
  3. In the window that pops up, navigate to .minecraft, saves
  4. Drag the downloaded .zip file into this folder

Now you have the world file, there are multiple ways you can set this up:

Host with a server
  1. Download the latest server.jar over at minecraft.net and launch it
  2. When the program closes, in the same directory open eula.txt
  3. Replace the false with true to accept the EULA
  4. Extract the .zip file and place the contents of it into a new folder called world
  5. Set-up 25565 port-forwarding on your router, more info @ portforward.com
  6. Once finished, launch up your server and share your IP that they can use to connect with!

Host with Realms

This requires an active subscription to Minecraft Realms on Java Edition

  1. Navigate to Minecraft Realms from the main menu and press the build button
  2. Choose one of the three presented world slots and select Reset world then Upload world
  3. Select the Haunting Hunters save from this menu and wait for it to be uploaded
  4. Once finished, invite your friends to the realm and have some fun!
